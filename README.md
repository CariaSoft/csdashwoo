# CSDashWoo v1.0 - WooCommerce Yönetim Paneli Eklentisi

CSDashWoo, WordPress ve WooCommerce tabanlı siteler için geliştirilmiş gelişmiş yönetim paneli özelleştirme eklentisidir. Admin panelini kullanıcı dostu ve iş odaklı hale getirerek, e-ticaret süreçlerinin daha verimli yönetilmesini sağlar.

## Özellikler

### 1. Menü Yönetimi
- Drag & drop ile kolay menü sıralama
- Roller bazlı menü erişimi
- Ana menü ve alt menü özelleştirme
- Menü düzen kilidi (sadece admin tarafından değiştirilebilir)

### 2. Bildirim Merkezi
- Yeni sipariş bildirimleri
- Sipariş durumu değişiklikleri
- Ürün yorumları
- Contact Form 7 mesajları
- Bildirim türlerini seçmeli olarak açma/kapama

### 3. Dashboard (WordPress kutucukları ile)
- Toplam satış özeti
- Bekleyen sipariş sayısı
- Açık işlemler
- Son bildirimler

### 4. Rol Bazlı Erişim
- **Administrator**: Tüm özelliklere tam erişim
- **Shop Manager**: Siparişler, ürünler, bildirimler, açık işler
- **Editor**: Ürünler ve yorumlar

## Kurulum

1. Eklentiyi WordPress admin paneline yükleyin
2. Eklentiyi etkinleştirin
3. Ayarlar > CSDashWoo menüsünden yapılandırmaları yapın

## Ayarlar

### Dashboard Ayarları
- Dashboard kutucuklarını açma/kapama
- Gösterilecek veri türlerini seçme

### Bildirim Ayarları
- Bildirim türlerini açma/kapama
- Teknik bildirimleri gizleme (varsayılan: açık)
- Contact Form 7 entegrasyonu (isteğe bağlı)

## Kullanım

### Menü Yöneticisi
1. Ayarlar > Menü Yöneticisi menüsüne gidin
2. Sol panelde mevcut menü yapısı, sağ panelde tüm menü öğeleri yer alır
3. Drag & drop ile menü sıralamasını düzenleyin
4. "Kaydet" butonuna tıklayarak değişiklikleri uygulayın

### Bildirimler
- Sağ üst köşedeki zil simgesi bildirimleri gösterir
- Bildirimler kategorilendirilmiş şekilde listelenir
- Her bildirim yeni/görüldü durumuna göre farklı şekilde gösterilir

## Çoklu Dil Desteği

Eklenti çoklu dil desteğine uygundur. "csdashwoo" text domaini kullanılır. Loco Translate gibi eklentilerle çeviri işlemleri yapılabilir.
